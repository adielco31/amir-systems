## 3. החלטות אסטרטגיות מרכזיות

### 3.1 מיצוב (Positioning)
**מסלול שנבחר:** "מומחה מולטימדיה שנותן גם שירותים נוספים", **לא** "מוסך טכני מקיף".

הסיבה: מולטימדיה לרכב היא נישה עם פחות תחרות באזור, יותר רווחיות, ואנשים מוכנים לנסוע אליה. מכונאות שגרתית — אנשים לא נוסעים בשבילה. דף הבית והשיווק יתמקדו במולטימדיה. שאר השירותים יקבלו עמודים מלאים (כדי לתפוס SEO) אבל לא יהיו הקטר.

### 3.2 ביטול אוריאה (AdBlue) — טיפול משפטי
**מסלול שנבחר:** ניסוח זהיר בשם "ייעוץ ופתרונות SCR / AdBlue".

הרקע המשפטי: ביטול AdBlue הוא הפרת תקנות פליטות (Euro 6). מתגלה מיידית במכון רישוי. עלול לגרור פסילת רישוי, קנסות, וביטול ביטוח. גוגל אוסרת על Google Ads עבור השירות. **לא לפרסם באתר ככותרת ברורה.**

הניסוח שלנו: "אבחון תקלות מערכת SCR / AdBlue, ייעוץ ופתרונות מותאמים". זה לוכד את ה-SEO ולא חושף משפטית.

**שם עמוד:** `/services/scr-adblue-consulting/`
**כותרת:** "אבחון תקלות SCR ו-AdBlue | אמיר מערכות"

### 3.3 עמודי מותגי רכב (Mazda, Toyota, וכו') — לא נבנים
ההחלטה: לא לבנות עמודים נפרדים למותגי רכב. הסיבה: עמודי מותג טובים דורשים תוכן ספציפי לכל דגם (התאמת חיישנים, אילוצים בחיווט). בלי תוכן ייחודי הם הופכים ל"thin content" שגוגל מענישה. עדיף לא לעשות אותם מאשר לעשות אותם בינוניים.

**אפשר לחזור לזה בעתיד** כשיש זמן ותוכן ספציפי לכל דגם.

### 3.4 תוכן בערבית (לנצרת) — תלוי החלטה
**סטטוס:** הוחלט שבשלב 1 יבנה עמוד נצרת בעברית בלבד.

הסיבה הפדגוגית-שיווקית: נצרת היא העיר הערבית הכי גדולה בישראל, ותחרות שיווק בערבית בנישה הזו כמעט אפס. עמוד אחד בערבית יכול לתפוס ~450 חיפושים חודשיים שאף מתחרה לא לוקח. **שאלה פתוחה:** האם אמיר/המשפחה דוברי ערבית או נכונים לשכור מתרגם?

- **כל עמוד = הזדמנות לדירוג בגוגל.** עמודים נפרדים לכל שירות חשובים גם אם התוכן דומה.

- **כל עמוד חייב 800+ מילים** של תוכן ייחודי. תוכן דליל = פגיעה ב-SEO.

## 5. סטאק טכנולוגי

| רכיב | בחירה | סיבה |
|------|--------|-------|
| Framework | **Astro 5** | Static site generation, אפס JS ב-default, מהירות 95+ ב-Lighthouse |
| Styling | **Tailwind v4** (Vite plugin) | Utility-first, RTL-aware, מהיר |
| Language | **JavaScript** (ללא TypeScript) | תואם להעדפות הצוות, פחות overhead |
| Hosting | **Vercel** או **Cloudflare Pages** | חינמי, edge CDN, אוטומטי-deploy מ-Git |
| Domain | **amirsystems.co.il** | .co.il נותן יתרון SEO לחיפושים בישראל |
| Images | **Cloudinary** (מומלץ) | אופטימיזציה אוטומטית, WebP, lazy loading |
| Analytics | **Google Analytics 4** + **Search Console** | חובה ל-SEO |
| Forms | **Web3Forms** או **Formspree** (חינמי) | טופס יצירת קשר ללא backend |
| Maps | **Google Maps Embed API** (חינמי) | מפה בדף יצירת קשר |
| Fonts | **Heebo** (Google Fonts) | פונט עברי מודרני, יציב |

### למה Astro ולא Next.js/React טהור
לאתר תוכן-מבוסס כמו זה, Astro מנצח את React Next.js פי 2 במהירות טעינה. שולח 0 JavaScript לדפדפן ב-default (במקום 200KB+ של React SPA), ויש לו את כל ה-SEO tooling מובנה. הסינטקס דומה ל-JSX ואפשר להטמיע רכיבי React אם צריך אינטראקטיביות.

## 7. אסטרטגיית SEO

### מילות מפתח מובילות (מהמחקר המקיף — `outputs/amir-systems-keyword-research.md`)

**Tier 1 — Primary (חייבות לתפוס):**
- מסך אנדרואיד לרכב — 2,400/חודש ארצי
- מולטימדיה לרכב — 1,900/חודש
- דאשקאם / מצלמת רכב — 3,300/חודש משולב
- מצלמת רוורס — 1,300/חודש
- חשמלאי רכב (+ עפולה) — 1,400/חודש ארצי, ~200 בעפולה
- התקנת וו גרירה — 900/חודש

**ה-5 כוכבים האסטרטגיים:**
1. **"כמה עולה התקנת מסך אנדרואיד לרכב"** — פוסט בלוג עם פוטנציאל 500+ ביקורים חודשיים
2. **"חשמלאי רכב עפולה"** — תחרות נמוכה, ROI גבוה, הכי קל לתפוס
3. **"מולטימדיה לרכב יד שנייה"** — סגמנט שאף מתחרה לא לוקח
4. **תוכן בערבית לנצרת** — הזדמנות אמיתית עם אפס תחרות
5. **שאלות "כמה עולה" באופן כללי** — מנוע תנועה גדול בנישה

### עקרונות SEO מנחים
- **NAP Consistency** — שם, כתובת, טלפון זהים בכל מקום (אתר, GBP, Facebook, ספרי עסקים)
- **Schema markup** — `LocalBusiness` בכל עמוד, `Service` בכל עמוד שירות, `FAQPage` בכל FAQ
- **תוכן ייחודי** — 800+ מילים לכל עמוד, ללא העתקות
- **תמונות אופטימיזציה** — WebP, lazy loading, alt text בעברית עם מילות מפתח
- **Mobile-first** — האתר חייב להיראות מצוין במובייל, רוב התנועה תגיע משם
- **Core Web Vitals** — LCP < 2.5s, FID < 100ms, CLS < 0.1

### GEO (Generative Engine Optimization)
- תוכן מובנה כשאלה-תשובה (FAQ בכל עמוד)
- רשימות ומדריכים ("5 הדברים לבדוק לפני...")
- סטטיסטיקות מקוריות ("85% מהלקוחות שלנו ב-2025 בחרו...")
- ציטוטים מותגים (Atoto, Pumpkin, Pioneer וכו׳ — להיות מצוטט)
